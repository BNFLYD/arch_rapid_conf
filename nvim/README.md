# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

This template is based on the Gentleman-Dots config files (https://github.com/Gentleman-Programming/Gentleman.Dots). Thanks to Alan-TheGentleman and all contributors for the base.

O en idioma OA, este template esta basado en los archivos de configuración de Gentleman-Dots (https://github.com/Gentleman-Programming/Gentleman.Dots). Gracias a Alan-TheGentleman y a todos sus colaboradores por la base.
